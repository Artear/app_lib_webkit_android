WebNavigationAction

 interface WebNavigationAction

An action that imply a possible navigation inside or outside your application. Maybe can be a filter to respond to a particular url.

WebNavigationActionManager

 class WebNavigationActionManager

A manager that have some actions to process. This actions should be added before load the url in WebWrapper. The wrapper has a client to override the loading of any url in the web view and use WebWrapper.overrideUrlLoading to pass the responsibility to this class to process that Uri.

WebNavigationActionPolicyDomain

 class WebNavigationActionPolicyDomain : WebNavigationAction

Works like a filter. Just add all domains that you want to manage, and this navigation action will intercept all other urls that does not contain in domains
