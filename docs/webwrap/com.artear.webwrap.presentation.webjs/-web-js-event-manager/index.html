WebJsEventManager

class WebJsEventManager

A web javascript event manager act like a mediator among your application and your WebView. Establish a communication between a CommandJs and each javascript event that you want to respond.

The Injector Library provides an extra functionality to WebWrapper. It's a annotation processor that generate an extension to this library creating all CommandJs with JsInterface.

JsEventManager is an annotation that provide an initialization for that CommandJs.
