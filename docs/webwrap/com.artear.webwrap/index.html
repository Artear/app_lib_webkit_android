WebWrapper

 class WebWrapper : LifecycleObserver

A wrapper of your webView. Manage the load of url and is a controller for url override executed in the web page across a navigation action. Also has enabled javascript and check all event from them using event js. The wrapper is a LifecycleObserver, and is lifecycle-aware.
